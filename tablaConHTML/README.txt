#Repasemos lo que hemos aprendido hasta ahora:

--------------------------------------------------------

-El <table> elemento crea una mesa.

-El <tr> elemento agrega filas a una tabla.

-Para agregar datos a una fila, puede usar el <td> elemento.

-Los títulos de las tablas aclaran el significado de los datos. Los encabezados se agregan con el <th> elemento.

-Los datos de la tabla pueden abarcar columnas utilizando el colspanatributo.

-Los datos de la tabla pueden abarcar filas utilizando el rowspanatributo.

-Las tablas se pueden dividir en tres secciones principales: un encabezado, un cuerpo y un pie de página.

-La cabecera de una tabla se crea con el <thead> elemento.

-El cuerpo de una tabla se crea con el <tbody> elemento.

-El pie de página de una tabla se crea con el <tfoot> elemento.

-Todas las propiedades de CSS que aprendió en este curso se pueden aplicar a las tablas y sus datos.
